# 轻历隐私说明

轻历核心版按“本地优先”设计。

## 不上传的数据

轻历不会上传姓名、生日、备注或使用记录。具体包括：

- 不上传生日联系人姓名。
- 不上传公历或农历生日日期。
- 不上传生日备注。
- 不上传提醒历史、使用频率、打开记录或其他使用记录。
- 不要求登录账号、会员身份或云同步。

## 本机保存的数据

生日数据库保存在：

```text
%LOCALAPPDATA%\QingLi\qingli.db
```

应用设置和本地运行状态也保存在 `%LOCALAPPDATA%\QingLi` 下。该目录属于当前 Windows 用户账号。

## 网络说明

当前核心版不依赖网络服务即可打开月历、管理生日和触发本地提醒。若未来加入同步、备份或在线节假日更新等功能，应在功能上线前更新本文档，并明确说明传输内容、开关和保留策略。

## 删除数据

退出轻历后，删除以下目录即可清除本机保存的轻历数据：

```text
%LOCALAPPDATA%\QingLi
```
