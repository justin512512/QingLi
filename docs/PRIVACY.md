# 轻历隐私说明

轻历核心版按“本地优先”设计。

## 不上传的数据

轻历不会上传姓名、生日、备注或使用记录。具体包括：

- 不上传生日联系人姓名或纪念日名称。
- 不上传公历或农历生日日期。
- 不上传生日、纪念日备注。
- 不上传提醒历史、使用频率、打开记录或其他使用记录。
- 不要求登录账号、会员身份或云同步。

## 本机保存的数据

生日和纪念日数据库保存在：

```text
%LOCALAPPDATA%\QingLi\qingli.db
```

应用设置和本地运行状态也保存在 `%LOCALAPPDATA%\QingLi` 下。该目录属于当前 Windows 用户账号。

## 网络说明

轻历不依赖网络即可打开日历、查看内置黄历和历史数据、管理提醒。可选数据更新只下载公开的节假日或历史事件包，不会发送姓名、生日、纪念日、备注、提醒历史或使用记录；下载包必须通过 HTTPS、版本和 SHA-256 校验。

## 删除数据

退出轻历后，删除以下目录即可清除本机保存的轻历数据：

```text
%LOCALAPPDATA%\QingLi
```
