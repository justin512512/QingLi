## 窗口布局覆盖

- 无记录时使用默认大小和任务栏附近位置。
- 支持顶部拖动和八个方向的边缘/边角缩放。
- 支持 100%/125%/150% 缩放、负坐标副屏和多显示器可见区约束。
- 支持布局持久化、并发写入、损坏数据回退和设置页重置。
- 原生定位或文件写入失败时不会导致主窗口崩溃，设置页会显示最近错误。

## Windows 11 实机建议

1. 先退出旧版轻历，再解压 v4 到新文件夹。
2. 验证单击轻历任务栏时钟可打开/隐藏日历。
3. 移动并调整窗口大小，重启后确认位置和大小保留。
4. 切换 125%/150% 缩放或副屏，确认窗口仍完全可见。
5. 在设置中点击“恢复默认窗口布局”，确认窗口回到默认布局。
6. 验证退出轻历后 Windows 系统时钟能正常恢复。
