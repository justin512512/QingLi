# 轻历：Windows 11 任务栏时钟与万年历设计规格

## 1. 产品定位

轻历是一款免费、纯本地运行的 Windows 11 桌面日历工具。它提供稳定的托盘时钟入口，并可在用户主动开启后隐藏 Windows 系统时钟，显示自定义任务栏时钟。点击时钟即可打开万年历，同时提供公历与农历生日提醒。

产品不包含账号、云同步、广告、会员或付费功能。核心目标是以较低的系统风险提供类似“优选日历”的常用能力。

## 2. 目标与成功标准

第一阶段交付应满足：

- 在 Windows 11 上安装并运行。
- 在通知区域提供稳定入口，点击后从任务栏右下角弹出日历。
- 显示公历、农历、二十四节气和中国法定节假日。
- 管理公历与农历生日，并为每条生日独立设置提前天数和提醒时间。
- 使用 Windows 通知发送提醒，休眠唤醒后补查遗漏提醒。
- 可选隐藏系统时钟并显示自定义时钟。
- 关闭替换模式或卸载软件后恢复 Windows 系统时钟。
- 所有个人数据仅保存在本机，日常功能无需联网。

## 3. 范围

### 3.1 第一版包含

- 通知区域图标及右键菜单。
- 任务栏自定义时钟。
- 月历与日期详情。
- 公历、农历、二十四节气及法定节假日。
- 公历和农历生日管理。
- 可配置的生日提醒。
- 开机启动、主题和显示格式设置。
- 本地数据库、异常恢复和单实例运行。

### 3.2 第一版不包含

- 天气预报。
- 黄历宜忌、星座和运势。
- 账号、云同步及跨设备同步。
- 手机端或其他桌面系统版本。
- 广告、会员和付费功能。
- 向 Explorer 进程注入代码或修改系统文件。

## 4. 技术方案

### 4.1 技术栈

- 语言与运行时：C#、.NET 8。
- 桌面界面：WPF。
- 数据存储：SQLite。
- 系统集成：Windows Win32 API、通知区域 API、Windows 通知。
- 目标平台：Windows 11 x64。
- 发布形式：自包含安装包，用户无需预装 .NET。

选择 WPF 是因为它适合轻量常驻桌面程序，能够成熟地处理无边框窗口、托盘入口、Win32 互操作、多显示器和高 DPI。

### 4.2 时钟模式

轻历提供两个层级：

1. **托盘模式（默认）**
   - 不修改系统时钟。
   - 通知区域显示轻历图标。
   - 点击图标打开日历。
   - 这是始终可用的安全入口。

2. **替换模式（用户主动开启）**
   - 使用 Windows 用户策略隐藏系统通知区域时钟。
   - 在任务栏右侧显示轻历的无边框时钟窗口。
   - 不向 Explorer 注入代码，不修改 Windows 系统文件。
   - 开启前记录原设置；关闭、卸载或恢复操作时还原。

替换模式属于“视觉替换”，不假设 Windows 提供第三方程序直接占用原生时钟控件位置的公开接口。若当前 Windows 构建、任务栏布局或策略不兼容，软件应保留托盘模式并明确提示，而不能强制修改系统。

## 5. 系统结构

### 5.1 应用外壳

职责：

- 管理应用启动、退出和单实例。
- 创建托盘图标和菜单。
- 管理日历、设置、生日编辑和自定义时钟窗口。
- 第二次启动时唤醒已有实例并显示日历。

### 5.2 日历引擎

职责：

- 生成公历月视图。
- 计算农历日期、生肖和二十四节气所需的基础日期信息。
- 合并法定节假日数据。
- 查询指定日期关联的生日。

该模块不依赖界面，便于针对跨年、闰月和边界日期进行单元测试。

### 5.3 生日与提醒服务

职责：

- 创建、修改、删除和查询生日。
- 根据生日类型计算每年的公历提醒日期。
- 按每条记录的提前天数和提醒时间生成提醒。
- 启动、跨日和休眠唤醒时补查未处理提醒。
- 防止同一提醒被重复发送。

### 5.4 系统集成服务

职责：

- 管理通知区域图标。
- 发送 Windows 通知。
- 管理开机启动。
- 检测任务栏位置、尺寸、显示器和 DPI。
- 启用、关闭及恢复系统时钟隐藏策略。
- 在 Explorer 重启或显示配置变化后重新定位自定义时钟。

### 5.5 本地存储

SQLite 至少包含：

- `birthdays`：姓名、日期类型、日期字段、是否闰月、提前天数、提醒时间、备注、启用状态。
- `reminder_history`：提醒记录、计划时间、发送时间和状态。
- `settings`：主题、时钟格式、每周首日、开机启动、替换模式等。
- `holiday_packages`：节假日数据版本及内容。
- `system_restore_state`：启用替换模式前的系统设置快照。

数据库存放在当前用户应用数据目录。日志不得记录完整生日备注等隐私内容。

## 6. 用户界面

### 6.1 任务栏时钟

- 默认两行显示：第一行时间，第二行月日与星期。
- 支持 24 小时和 12 小时格式。
- 支持调整字号、文字颜色和日期格式。
- 自动跟随 Windows 浅色或深色主题，也允许手动选择。
- 单击打开或关闭日历。
- 右键打开快捷菜单。

### 6.2 弹出日历

- 从当前任务栏所在屏幕的右下角弹出。
- 默认显示当月，支持前后月份和“回到今天”。
- 今天使用明显但克制的强调色。
- 工作日、周末、法定节假日、调休工作日和生日使用不同标识。
- 选中日期后显示公历、农历、节气、节假日和生日摘要。
- 点击窗口外部或按 Esc 关闭。

### 6.3 生日管理

字段包括：

- 姓名。
- 公历或农历。
- 出生日期。
- 农历闰月标记。
- 提前提醒天数。
- 提醒时间。
- 备注。
- 是否启用。

列表支持按姓名搜索和按最近生日排序。删除操作需要二次确认。

### 6.4 设置

- 开机启动。
- 启用或关闭替换系统时钟。
- 自动、浅色或深色主题。
- 时间和日期格式。
- 一周从周一或周日开始。
- 打开数据目录。
- 恢复系统时钟。
- 查看软件版本与本地数据说明。

### 6.5 托盘菜单

- 打开日历。
- 添加生日。
- 打开设置。
- 暂停今日提醒。
- 恢复系统时钟。
- 退出。

## 7. 数据与事件流

### 7.1 启动流程

1. 检查是否已有实例。
2. 加载设置和数据库。
3. 校验上次替换模式的系统恢复状态。
4. 创建托盘入口。
5. 若已启用替换模式，检查兼容性并创建自定义时钟。
6. 启动提醒调度并补查遗漏提醒。

### 7.2 点击时钟

1. 获取点击所在任务栏和显示器。
2. 查询当月日历数据。
3. 合并农历、节气、节假日和生日标记。
4. 在任务栏上方显示无边框日历窗口。

### 7.3 生日提醒

1. 根据生日记录计算当前年度发生日期。
2. 减去提前天数并组合提醒时间。
3. 调度器在到期时发送 Windows 通知。
4. 写入提醒历史，避免重复。
5. 用户点击通知后打开对应生日或日期详情。

## 8. 法定节假日数据

第一版内置可用年度的中国法定节假日数据，包含：

- 节日名称。
- 放假日期。
- 调休工作日。
- 数据适用年份和版本。

法定安排可能每年变化，因此日历引擎必须把节假日作为独立数据包处理，不能把安排散落硬编码在界面代码中。首版提供本地数据包更新入口；联网自动更新不属于首版要求。

## 9. 农历生日规则

- 普通农历生日按每年对应农历月日换算为公历。
- 对于“闰月生日”，记录是否指定闰月。
- 当某年不存在对应闰月时，默认在同月非闰月日期提醒，并在详情中注明换算规则。
- 若目标农历月没有记录中的日期，例如小月不存在三十日，则默认使用该月最后一天。
- 所有换算逻辑必须独立测试，界面不得自行计算。

## 10. 提醒行为

- 每条生日可单独设置提前天数和提醒时间。
- 同一生日可在设定时间产生一次提醒。
- 应用启动、跨日、系统时间变化和休眠唤醒时重新校准。
- 若计划时间已过但仍处于当天，立即补发一次。
- 若提醒日期已经完全过去，不补发陈旧通知。
- “今天不再提醒”只抑制当天后续提醒，不改变生日配置。

## 11. 故障处理与恢复

### 11.1 替换模式保护

- 修改系统时钟显示设置前保存原值和“是否原本存在”的状态。
- 设置变更失败时不创建替代时钟，并显示可理解的错误信息。
- 自定义时钟无法启动或反复崩溃时恢复系统时钟。
- 设置页和托盘菜单始终提供“恢复系统时钟”。
- 卸载程序执行恢复操作；若卸载恢复失败，应提供独立恢复命令或小型恢复工具。

### 11.2 数据保护

- 数据库写入采用事务。
- 数据库损坏时保留原文件并尝试只读打开，不静默覆盖。
- 用户输入进行日期与范围校验。
- 不依赖网络即可查看和管理生日。

### 11.3 Explorer 与显示变化

- 监听 Explorer/任务栏重建、显示器连接变化、分辨率变化和 DPI 变化。
- 自定义时钟失去合理位置时自动重新定位。
- 无法定位时隐藏自定义时钟并保留托盘入口。

## 12. 测试策略

### 12.1 自动化测试

- 公历与农历日期换算。
- 二十四节气边界日期。
- 农历跨年生日。
- 闰月存在和不存在的年份。
- 农历月无三十日的回退规则。
- 提前提醒日期计算。
- 休眠唤醒后的补查与去重。
- 设置和系统恢复状态的读写。

### 12.2 集成测试

- 托盘图标创建、点击和右键菜单。
- Windows 通知发送与点击。
- 开机启动。
- 隐藏及恢复系统时钟。
- Explorer 重启后恢复。
- 单实例激活。

### 12.3 手工兼容测试

- Windows 11 当前受支持版本。
- 100%、125%、150% 和 200% 缩放。
- 单显示器和多显示器。
- 底部任务栏及系统允许的任务栏布局。
- 浅色、深色和高对比度主题。
- 休眠、唤醒、锁屏及跨日。

## 13. 发布阶段

### 阶段一：稳定核心

- 托盘入口。
- 万年历。
- 本地生日管理。
- 提醒。
- 设置与开机启动。

### 阶段二：任务栏替换

- 系统时钟隐藏与恢复。
- 自定义任务栏时钟。
- 多显示器、高 DPI、Explorer 重启适配。
- 独立恢复能力。

先完成阶段一可保证软件即使在特定 Windows 11 构建上无法替换系统时钟，仍然是一款完整可用的日历和生日提醒工具。

## 14. 验收清单

- 首次启动无需登录或联网。
- 托盘入口可稳定打开日历。
- 月历正确显示公历、农历、节气和已内置的法定节假日。
- 公历与农历生日可新增、编辑、删除并正确标记。
- 每条生日可设置独立的提前天数和提醒时间。
- 到期提醒只发送一次，休眠唤醒后不会漏掉当天提醒。
- 开启替换模式后可显示自定义任务栏时钟。
- 关闭替换模式、异常恢复或卸载后，Windows 系统时钟可恢复。
- 软件不修改系统文件，不向 Explorer 注入代码。
- 个人数据仅保存在本机。
