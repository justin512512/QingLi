# 轻历用户指南

## 打开日历

启动轻历后，单击任务栏右下角由轻历显示的时间和日期区域。再次单击、按 `Esc` 或点击其他窗口可隐藏日历。

轻历使用安全覆盖层接管点击区域，不修改 Windows 系统文件，也不向 Explorer 注入代码。

三栏内容分别是：

- 左侧：历史上的今天，单击条目可打开来源页面。
- 中间：所选日期、农历、干支、生肖、宜忌、节气、节日以及 42 格月历。
- 右侧：未来 90 天的节假日、节气、传统节日、生日和纪念日倒计时。

键盘可用方向键切换日期，`PageUp`/`PageDown` 切换月份，`Home` 返回今天。

## 移动、缩放和恢复布局

- 按住日历顶部拖动区即可移动窗口。
- 拖动窗口的四条边或四个角可自由调整大小，最小为 760×420。
- 窗口位置和大小会自动记住，多显示器或缩放比例变化后会自动约束到可见工作区。
- 如需重置，打开“设置”，点击“恢复默认窗口布局”。

布局数据保存在：

```text
%LOCALAPPDATA%\QingLi\calendar-window-layout.json
```

## 生日和纪念日

右下角的“＋生日”和“＋纪念日”会自动带入当前选中日期。支持公历、农历、闰月、提前提醒天数、提醒时间和备注。数据保存在：

```text
%LOCALAPPDATA%\QingLi\qingli.db
```

## 恢复 Windows 时钟

可在托盘菜单或设置中关闭任务栏时钟替换。若轻历异常退出后系统时钟没有恢复，请运行同目录中的：

```text
QingLi.Recovery.exe --restore-clock
```

详细步骤见 `docs/CLOCK-RECOVERY.md`。
