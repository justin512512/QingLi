# 轻历用户指南

## 打开日历

启动轻历后，单击任务栏右下角由轻历显示的时间和日期区域。再次单击、按 `Esc` 或点击其他窗口可隐藏日历。轻历使用安全覆盖层接管点击区域，不修改 Windows 系统文件，也不向 Explorer 注入代码。

三栏内容分别是：

- 左侧：历史上的今天，单击条目可打开来源页面。
- 中间：所选日期、农历、干支、生肖、宜忌、节气、节日以及 42 格月历。
- 右侧：未来 90 天的节假日、节气、传统节日、生日和纪念日倒计时。

键盘可使用方向键切换日期，`PageUp`/`PageDown` 切换月份，`Home` 返回今天。

## 生日和纪念日

右下角的“＋生日”和“＋纪念日”会自动带入当前选中日期。两者均支持公历、农历、闰月、提前提醒天数、提醒时间和备注。生日与纪念日只保存在：

```text
%LOCALAPPDATA%\QingLi\qingli.db
```

## 离线与更新

黄历、节假日和 366 天历史事件随程序提供，无网也能使用。可选更新包只包含公开日历数据，必须通过 HTTPS、版本、SHA-256 和格式校验；失败时自动继续使用上一版本或内置版本。

## 恢复 Windows 时钟

可在托盘菜单或设置中关闭任务栏时钟替换。若轻历异常退出后系统时钟没有恢复，请运行同目录中的：

```text
QingLi.Recovery.exe --restore-clock
```

详细步骤见 `docs/CLOCK-RECOVERY.md`。
